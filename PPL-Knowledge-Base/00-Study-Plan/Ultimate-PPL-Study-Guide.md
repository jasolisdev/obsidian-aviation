# Ultimate PPL Study Guide

Use this as a primary study roadmap for any PPL learner. It is organized for three goals at once:
1. Pass the FAA knowledge test.
2. Be strong in every oral exam area.
3. Fly safely and confidently to private pilot standards.

## 1. Start Here (Order of Operations)

1. Read this page once end-to-end.
2. Open [[00-Study-Plan/FAA-Current-References-2026|FAA Current References (2026 Baseline)]].
3. Pick a path from [[00-Study-Plan/Study-Track-Selector|Study Track Selector]].
4. Run the selected track as your weekly execution board.
5. Study from one concept index per day:
   - [[01-Regulations/Concepts|Regulations]]
   - [[02-Aerodynamics/Concepts|Aerodynamics]]
   - [[03-Aircraft-Systems/Concepts|Aircraft Systems]]

6. Finish each session with 10-20 oral questions from [[08-Oral-Exam/Oral-QA|Oral Q&A]].
7. Track weak areas in [[08-Oral-Exam/Trick-Questions|Trick Questions]].
8. Update [[00-Study-Plan/Progress|Progress]] weekly.

## 2. How To Study Each Note (High Retention)

For each concept note, do this in sequence:
1. Read the definition out loud.
2. Explain why it matters operationally in your own words.
3. Redraw the diagram from memory.
4. Answer the oral questions without looking.
5. Tie the concept to one real scenario in a training aircraft.

If you cannot do all 5 quickly, keep the note in `seed` or `refining` status.

## 4. Weekly Cadence (Sustainable)

- `3 days/week`: new concepts (1-2 notes/day).
- `2 days/week`: oral drills + scenario questions.
- `1 day/week`: chart/weather decoding only.
- `1 day/week`: full review + update `Progress`.

Use [[00-Study-Plan/Weekly-Plan|Weekly Plan]] as your execution board.

## 5. Knowledge Test Strategy (FAA PAR)

1. Use ACS-aligned studying, not random memorization.
2. When you miss a practice question, record the topic and ACS element in `Trick Questions`.
3. Re-study from FAA source first, then your note.
4. Re-test only missed domains before taking full mocks.

## 6. Oral Exam Strategy (DPE-Style)

For each answer, aim for this 4-part structure:
1. Rule/concept.
2. Operational meaning.
3. Example in a training airplane.
4. Risk decision if conditions worsen.

If any answer is purely mnemonic with no operational context, it is not checkride-ready.

## 7. Flight Lesson Integration

Before each lesson:
1. Read 2-3 notes tied to that lesson.
2. Brief one scenario you may encounter.
3. Identify one regulation and one performance item to verify.

After each lesson:
1. Add one correction or insight to the note.
2. Add one oral question from what you missed.

## 8. Non-Negotiable Accuracy Rules

- Treat FAA publications and current regulations as final authority.
- If a note conflicts with FAA text, fix the note immediately.
- Recheck date-sensitive items monthly:
  - ACS version
  - AIM effective date
  - Chart cycle
  - Handbook addenda

Use [[00-Study-Plan/FAA-Current-References-2026|FAA Current References (2026 Baseline)]] for this.

## 9. Finish-Line Checklist (Ready for Checkride)

- [ ] You can explain private pilot privileges/limits without guessing.
- [ ] You can compute W&B and performance from POH data.
- [ ] You can decode current METAR/TAF and defend a go/no-go call.
- [ ] You can brief airspace requirements from a sectional in under 2 minutes.
- [ ] You can run through emergency priorities from memory (aviate, navigate, communicate).
- [ ] You can answer oral scenarios with legal + operational + risk logic.
