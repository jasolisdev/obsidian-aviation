# Self-Paced Track

A flexible track for learners with inconsistent weekly availability.

## Suggested duration
- 14-20 weeks total.
- 3-4 study sessions per week.
- 1 oral drill block per week.

## Phase structure

### Phase 1 (Weeks 1-4)
- Regulations + Human Factors foundation.
- Build legal fluency and decision frameworks.

### Phase 2 (Weeks 5-8)
- Aerodynamics + Aircraft Systems.
- Focus on control, energy, and system-failure logic.

### Phase 3 (Weeks 9-12)
- Weather + Navigation.
- Emphasis on go/no-go, charting, communication, and diversion.

### Phase 4 (Weeks 13-16+)
- Performance/W&B + full oral integration.
- Run mock oral sessions and tighten weak areas.

## Weekly template

1. Session A: new concept notes.
2. Session B: regulations/weather scenario drill.
3. Session C: oral Q&A + weak-area remediation.
4. Optional Session D: catch-up and diagram redraw.

## Completion standard

- Can answer 50 random oral prompts at >85%.
- Can complete a full cross-country planning brief end-to-end.
- Can explain legal + operational + risk logic for common scenarios.
