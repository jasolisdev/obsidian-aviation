# PHAK Chapter 15: Navigation

How to get from Point A to Point B without getting lost or violating airspace. This chapter covers the tools (Charts) and the systems (VOR/GPS).

## Key Concepts
- **Pilotage vs. Dead Reckoning:** Looking out the window vs. trusting the math.
- **Charts:** Latitude/Longitude, Sectionals, Chart Supplements.
- **Radio Nav:** VOR principles, DME.
- **GPS:** Satellites, RAIM, WAAS.
- **Lost Procedures:** The 5 Cs.
